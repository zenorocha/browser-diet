Pra que usar o método `attr()` para buscar o ID?

```js
$('a').on('click', function() {
  console.log( $(this).attr('id') );
});
```

Se você pode buscar esse atributo nativamente através do `this`.

```js
$('a').on('click', function() {
  console.log( this.id );
});
```

E ainda ser mais rápido.
